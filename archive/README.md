# Archive - Boxing Round Splitter

Ce dossier contient les documents et tâches terminés ou obsolètes du projet.

## Structure

```
archive/
├── README.md                # Ce fichier
├── TODOS/                   # Tâches complétées ou obsolètes
└── [autres dossiers]        # Autres documents archivés
```

## Contenu Actuel

- **TODOS/** : Contient les fichiers TODO et STATUS des tâches complétées ou obsolètes

## Règles d'Archivage

1. **Quand archiver** :
   - Une tâche est complétée
   - Un document devient obsolète
   - Une décision est supersédée

2. **Comment archiver** :
   - Déplacer les fichiers vers le sous-dossier approprié
   - Mettre à jour `/docs/TIMELINE.md` avec la date et la raison
   - Conserver la structure originale des fichiers

3. **Organisation** :
   - Les TODOs complétés vont dans `archive/TODOS/`
   - Les documents obsolètes vont dans des sous-dossiers appropriés
   - Toujours inclure un README.md expliquant le contenu

## Historique

Consultez `/docs/TIMELINE.md` pour l'historique complet des archivages.
