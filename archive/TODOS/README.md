# Archive des TODOs - Boxing Round Splitter

Ce dossier contient les fichiers TODO et STATUS des tâches complétées ou obsolètes.

## Structure

```
archive/TODOS/
├── README.md            # Ce fichier
├── XX_nom_tache.md      # Fichiers TODO archivés
└── XX_nom_tache_STATUS.md # Fichiers STATUS archivés
```

## Contenu

Les fichiers dans ce dossier suivent la convention de nommage :
- `XX_nom_tache.md` : Fichier TODO original
- `XX_nom_tache_STATUS.md` : Fichier STATUS associé

## Règles

1. **Ne pas modifier** les fichiers archivés - ils représentent l'état historique
2. **Conserver les paires** TODO+STATUS ensemble
3. **Référence** : Ces fichiers peuvent être consultés pour comprendre l'historique du projet

## Historique

Consultez `/docs/TIMELINE.md` pour voir quand et pourquoi chaque TODO a été archivé.
