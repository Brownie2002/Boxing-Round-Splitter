# TODOs Actifs - Boxing Round Splitter

Ce dossier contient les fichiers TODO et STATUS des tâches actuellement actives dans le projet.

## Structure

```
TODOS/
├── README.md                # Ce fichier
├── XX_nom_tache.md         # Fichiers TODO actifs
└── XX_nom_tache_STATUS.md   # Fichiers STATUS associés
```

## Convention de Nommage

- **Fichiers TODO** : `XX_nom_tache.md` (où XX est un numéro séquentiel)
- **Fichiers STATUS** : `XX_nom_tache_STATUS.md` (correspondant au TODO)

## Cycle de Vie

1. **Création** : Un nouveau TODO est créé avec un fichier STATUS vide
2. **Exécution** : Le STATUS est mis à jour régulièrement
3. **Complétion** : Quand terminé, le TODO est déplacé vers `/archive/TODOS/`

## Règles

1. **Un TODO = Une tâche spécifique** avec des sous-tâches claires
2. **Un STATUS = Suivi de progression** avec pourcentage et checklist
3. **Pas de dates** dans les fichiers (l'historique est dans TIMELINE.md)
4. **Références claires** aux documents liés si nécessaire

## Tâches Actuelles

Consultez les fichiers individuels pour voir les détails des tâches en cours.

## Historique

Les TODOs complétés sont archivés dans `/archive/TODOS/` et référencés dans `/docs/TIMELINE.md`.
